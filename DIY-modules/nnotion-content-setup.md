# Notion 內容工作空間建立指南

安裝 Claude Code 之後，需要在你的 Notion 建立三個資料庫，讓 AI 助理可以直接幫你記錄和追蹤。

---

## 資料庫 1：靈感原料庫

**欄位設定：**

| 欄位名稱 | 類型 | 選項 |
|---------|------|------|
| 靈感標題 | 標題 | |
| 來源類型 | 單選 | 競爭者觀察 / 客戶諮詢 / 朋友討論 / 自我反思 / 網路收集 |
| 漏斗位置 | 單選 | 引流 / 培育 / 成交 |
| 潛力 | 單選 | 🔥 高 / ⚡ 中 / 💤 低 |
| 靈感描述 | 長文字 | |
| 記錄日期 | 日期 | |
| 已發展成的內容 | 關聯 | → 社群媒體規劃系統 |

---

## 資料庫 2：社群媒體規劃系統

**欄位設定：**

| 欄位名稱 | 類型 | 選項 |
|---------|------|------|
| 貼文主題 | 標題 | |
| 平台 | 單選 | IG / Threads / YouTube |
| 漏斗位置 | 單選 | 引流 / 培育 / 成交 |
| 狀態 | 狀態 | 構思中 / 撰寫中 / 待發布 / 已發布 |
| 預定發布日期 | 日期 | |
| CTA | 短文字 | |
| 內容草稿 | 長文字 | |
| 靈感來源 | 關聯 | → 靈感原料庫 |

---

## 資料庫 3：成效追蹤 Dashboard

**欄位設定：**

| 欄位名稱 | 類型 | 說明 |
|---------|------|------|
| 週次 | 標題 | 例如：2026 W16 |
| 週報日期 | 日期 | |
| 發布篇數 | 數字 | 這週實際發了幾篇 |
| 目標篇數 | 數字 | 這週目標幾篇 |
| 總觸及 | 數字 | 所有貼文觸及加總 |
| 總互動 | 數字 | 留言 + 分享 + 收藏 |
| 粉絲數變化 | 數字 | 本週增加幾個 |
| 本週觀察 | 長文字 | 什麼內容表現好、什麼不好 |
| 下週重點 | 長文字 | 根據這週調整方向 |

---

## 建好之後

把三個資料庫的連結填入你的 `CLAUDE.md` 的 Notion 工作空間區塊。

之後說「記錄靈感」「新增貼文」「看本週進度」，Claude 就會直接幫你操作這三個資料庫。
