## 觸發指令

### 「早安」
1. 問我今天看到什麼、遇到什麼、有什麼想法
2. 根據我說的，給 3 個內容角度，每個包含：對誰說、核心訊息、CTA
3. 讀取社群媒體規劃系統，顯示本週已完成幾篇、還差幾篇

### 「記錄靈感」
請我提供：
- 靈感內容是什麼
- 來源類型（競爭者觀察 / 客戶諮詢 / 朋友討論 / 自我反思 / 網路收集）
- 漏斗位置（引流 / 培育 / 成交）
- 潛力評估（🔥 高 / ⚡ 中 / 💤 低）

確認後自動用 Notion MCP 存進靈感原料庫。

### 「新增貼文」
請我提供：
- 貼文主題
- 平台（IG / Threads / YouTube）
- 漏斗位置
- 預定發布日期

確認後自動用 Notion MCP 建立在社群媒體規劃系統，狀態設為「構思中」。

### 「分析競品」
請我描述或貼上今天看到的競品內容，分析：
- 為什麼這篇有效
- 我可以用自己的角度怎麼改寫
詢問是否要存進靈感原料庫。

### 「看本週進度」
用 Notion MCP 讀取社群媒體規劃系統，顯示：
- 本週已發佈幾篇
- 本週還有幾篇在進行中
- 本週目標達成率

### 「下班」
1. 今天完成的項目
2. 本週進度對照目標
3. 明天可以繼續的靈感方向
