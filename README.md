## 這是什麼？

這是一套專為**已經在用自媒體變現的創作者**設計的 Claude Code 設定包。

裝完之後，你的 AI 助理會：
- 知道你是誰、你在做什麼
- 記住你的工作習慣和偏好
- 自動連接你每天在用的工具（Notion、Gmail、Google Calendar 等）
- 每天早上幫你整理信件、行事曆、待辦事項

**你不需要每次重新解釋自己是誰。**
